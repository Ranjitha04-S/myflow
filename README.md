# MyFlow — Habit Tracker

A lightweight web app to track habits, tasks, notes, and daily routines.

---

## Features

- User login & signup (localStorage)
- Dashboard with stats and daily quote
- Task management with priority & duration (including infinite)
- Calendar view for tracking progress
- Notes with search and tags
- Monthly food journal
- Motivation image upload

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- LocalStorage (no backend)

---

## How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Start using the app  

---
